Apa Itu Binary Tree?

Struktur data penting dalam ilmu komputer

Pengertian Binary Tree

Binary Tree (Pohon Biner) adalah struktur data berbentuk pohon di mana setiap node hanya memiliki paling banyak dua anak, yaitu anak kiri dan anak kanan. Binary Tree digunakan dalam berbagai aplikasi seperti pencarian data, pengurutan, kompresi data, dan lainnya.

Struktur Dasar


class Node:
    def __init__(self, data):
        self.data = data
        self.left = None
        self.right = None

Setiap node menyimpan data dan dua referensi: satu ke anak kiri dan satu ke anak kanan.

Jenis-Jenis Binary Tree

  • Full Binary Tree: Setiap node memiliki 0 atau 2 anak.
  • Complete Binary Tree: Semua level penuh kecuali mungkin level terakhir.
  • Perfect Binary Tree: Semua node internal punya 2 anak dan semua daun di level yang sama.
  • Binary Search Tree (BST): Kiri < node < kanan.

Kelebihan & Kegunaan

  • Meningkatkan efisiensi pencarian dan penyisipan data.
  • Dasar dari struktur data lanjutan seperti AVL Tree, Red-Black Tree, dan Heap.
  • Berguna dalam parsing ekspresi, pengkodean Huffman, dan lainnya.
